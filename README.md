# 🗳️ Pakistan Electronic Voting System

A professional electronic voting system developed in Python for the Election Commission of Pakistan. This system provides secure and efficient digital voting with comprehensive voter management and real-time result tracking.

## 📋 Features

### Voter Features
- **Voter Registration** - Register voters with CNIC verification
- **Secure Voting** - One CNIC, one vote mechanism
- **Live Results** - Real-time election results with percentage bar charts
- **Candidates List** - View all candidates with their parties and symbols
- **Voter Search** - Search registered voters by CNIC

### Admin Features
- **Election Control** - Start/Stop election process
- **Statistics Dashboard** - View voter turnout and election statistics
- **Vote Reset** - Reset all votes (requires confirmation)
- **Password Management** - Change admin password
- **Voter Management** - View complete registered voters list
